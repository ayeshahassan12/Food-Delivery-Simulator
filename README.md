# Food Delivery Simulator

A console-based Food Delivery Simulator developed in C++. The project simulates a simple food ordering system where users can browse menu items, place orders, and generate bills.

## Features

- Customer Menu
- Browse Food Items
- Place Orders
- Order Summary
- Bill Calculation
- Console-Based Interface

## Technologies Used

- C++
- Object-Oriented Programming (OOP)
- Visual Studio Code

## Concepts Applied

- Classes and Objects
- Functions
- Conditional Statements
- Loops
- Arrays / Vectors (depending on implementation)

## Learning Outcomes

- Strengthened C++ programming skills
- Applied OOP concepts in a real-world project
- Improved problem-solving and program design
